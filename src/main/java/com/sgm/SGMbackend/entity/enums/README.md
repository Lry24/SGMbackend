# Enums Package

Contient toutes les énumérations typant les données du système.

## Enums Principaux

- **Role**: AGENT, ADMIN (Sécurité).
- **StatutDepouille**: ADMIS, SORTI, EN_AUTOPSIE (Cycle de vie).
- **StatutChambre**: LIBRE, OCCUPEE, MAINTENANCE.

## Usage

Les enums garantissent l'intégrité des types en base de données et dans la logique métier.
