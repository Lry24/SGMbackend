# Mapper Package

Gère la conversion bidirectionnelle entre les Entités et les DTOs.

## Technologie

Utilise **MapStruct** pour la génération automatique du code de mapping.
Les interfaces définissent les mappings et MapStruct génère les implémentations lors de la compilation.

## Intégration

Les mappers sont injectés en tant que composants Spring dans les services.
